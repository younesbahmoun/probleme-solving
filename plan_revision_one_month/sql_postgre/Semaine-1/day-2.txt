● Questions Théoriques :
    1)
        - différence entre DDL (CREATE, ALTER) et DML (INSERT, UPDATE,DELETE):
            - DDL et (Data Definition Language) et Commandes SQL pour definer des tables et objets de la base de données (CREATE TABLE, ALTER TABLE, DROP TABLE)
            - DML (Data Manipulation Language) et commendes SQL pour manipuler data de la base de données (insertion, modification, suppression, lecture)
    2)
        -  une clé primaire ceat'un concept utilise en base de données pour separer le row par id caracteristique et unique et not null et auto increment
        -  chat: 
            -  Une clé primaire (Primary Key) est un attribut ou un ensemble d’attributs qui identifie de manière unique chaque enregistrement dans une table.
    3)
        - filtrage de données avec WHERE par id ou nom
        - chat:
            -  La clause WHERE permet de sélectionner uniquement les enregistrements qui remplissent une condition donnée dans une requête SQL.
            -  Elle agit comme un filtre sur les lignes d’une table.
            -  Seules les lignes vérifiant la condition sont traitées ou renvoyées.
